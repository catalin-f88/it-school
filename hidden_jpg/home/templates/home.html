Welcome to Hidden JPG!

With this application you can encrypt or decrypt secret messages hidden in .JPG files. To encrypt a secret message inside an image follow these steps:

  • Click the ENCRYPT button
  • Select the image you want to upload
  • Write the message you want to hide inside
  • Click the ENCRYPT AND DOWNLOAD FILE button

You will be given back the same image but with a hidden message inside.

The image will work the same as a normal image and if you upload it in the DECRYPT section you can see the message hidden inside

In order to decrypt a message hidden inside a picture, follow these steps:

  • Click the DECRYPT button
  • Select the image you know it has a hidden message
  • Click the DECRYPT HIDDEN MESSAGE button
