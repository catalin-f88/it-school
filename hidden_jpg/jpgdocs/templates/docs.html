Hidden JPG App Documentation

With this application you can encrypt or decrypt secret messages hidden in .JPG files. This method of transmiting message is called STEGANOGRAPHY.

Steganography is the art and science of writing hidden messages in such a way that no one, apart from the sender and intended recipient, suspects the existence of the message, a form of security through obscurity. Steganography is often used with cryptography.

This application is able to both ENCRYPT and DECRYPT messages in .JPG image files.

In order to start your journey, please click the ENCRYPT or DECRYPT buttons!

After you encrypt a hidden message inside an image, you can send that image to a friend and the file will work as intended (as an image!). Only if the file is uploaded in the application's DECRYPT section, the hidden message will be releaved!

At this time, the application only works on .JPG and .JPEG files, but more extensions will be added in the future!
